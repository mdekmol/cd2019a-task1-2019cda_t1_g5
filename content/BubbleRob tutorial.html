BubbleRob tutorial

操作步驟 :

1.

依照路徑[Menu bar --> Add --> Primitive shape --> Sphere]新增一個直徑為 : 0.2 m的球體，記得檢查"Create dynamic and respondable shape"是有被勾選的。

2.

按下ok後，球體便會新增在畫面中，為測試球體是可以調整參數以及可以作動的，<control-space>開始模擬，<control-c>複製球體後再按<control-v>會看到兩球體相撞後各自滾開，驗證此球體是可以作動的。

3.

停止模擬，雙擊球體，打開編輯視窗，將Collidable, Measurable, Renderable and Detectable 選項勾選

4.

接著，點選球體，打開位置視窗，切換到translation，在Z軸上打上0.02，確定Relative to是在World 後，點擊Translate selection，球體就會往上0.02 m

5.

[Menu bar --> Add --> Proximity sensor --> Cone type]新增距離感測器，打開方向視窗，切換到Rotation，在Y.Z軸打上90度，按下確定將感測器轉至側邊。

6.

打開位置視窗，在X及Z軸輸入0.1及0.12將感測器對準球體中心

7.

雙擊感測器圖示，進到編輯視窗，點擊Show volume parameter按鈕，輸入數據如下:

Offset : 0.005

Angle : 30 

Range : 0.15

完成後關閉此視窗，點擊Show detection parameters按鈕，取消Don't allow detections if distance smaller than的勾勾。

8.

選取球體及感測器[Menu bar --> Edit --> Make last selected object parent]

會變成這樣:

9.

 [Menu bar --> File --> New scene]新增一個場景之後，新增一個（0.08,0.08,0.02）的圓柱體

10.

選擇主體檢查一次步驟三的勾選，之後更改主體的絕對位置及絕對方向，改完後複製一個副本將絕對位置Y座標設 -0.1

 11.

將兩個做好的輪子複製，回原本場景貼上。

再來要做接頭 [Menu bar --> Add --> Joint --> Revolute]，保持選擇的接頭再選擇左輪，在位置對話框點擊 Apply to selection ，在方向對話框點擊 Apply to selection ；右輪的設定重複一次設定左輪的方式。設定完的圖示如下，接頭會定位在輪子中心

12.

在新場景建立一個直徑0.05的球體(滑塊)，一樣讓它具有 Collidable，Measurable，Renderable和Detectable ，[Menu bar --> Add --> Force sensor]添加一個力傳感器並向上移動0.05，將滑塊連接到力傳感器，將兩個物體複製回原場景並沿X軸移動 -0.07連到機器人本體。

13.

選擇兩個輪子和滑塊，在形狀動態對話框中點擊三次M = M * 2

14.

[Menu bar --> Tools --> Collections] 打開集合對話框，點Add new collection ，選擇新的集合項目為bubbleRob，點 Add ，幫它改名為 bubbleRob_collection。

15.

[Menu bar --> Tools --> Calculation module properties]打開計算模塊屬性對話框，點Add new distance object，選擇新的集合項目bubbleRob_collection，重命名為bubbleRob_distance。

16.

這時我們按下開始模擬，並不會看到有任何改變，但實際上小汽車上的感測器已經開始感測小汽車與可備感測物體之間的最小距離，但問題是畫面中並無可備感測的物體，因此在下一步驟我們要新增可備感測的物體。

17.

這一步要新增一個用來顯示物體與小汽車距離的圖表，[Menu bar --> Add --> Graph]，新增後將之改名為"bubbleRob_graph"，將graph拉到bubbleRob底下並設定graph的位置再(0，0，0.005)

18.

接下來雙擊bubbleRob_graph，取消打勾"Display XYZ-planes"然後點擊Add new data stream to record

Data stream type  :  absolute x-position

Object / item to record  :  bubbleRob_graph

bubbleRob_graph's absolute x-coordinate便會出現在 Data stream recording list裡面，照上述步驟新增y跟z，會長這樣 :

19.

在來設置3D 的曲獻，顯示小汽車的行走軌跡，點擊Edit 3D curves，接著點Add new curve，在對話框哩，如下設定 :

 X-value : bubbleRob_x_pos

Y-value : bubbleRob_y_pos

Z-value :  bubbleRob_z_pos

設定好之後，在最底下

點選 Relative to world 以及設定 Curve width 為 4

20.

這步驟要新增障礙物，新增primitive cylinder，尺寸為(0.1，0.1，0.2)，並解將這些選項打勾 Collidable, Measurable, Renderable, Detectable

複製這些圓柱，將小汽車圍住即可。

21.

雙擊bubbleRob，切換到common然後勾選Object is model base 與Object/model can transfer or accept DNA ，完成後再把兩個轉軸(Joint)隱藏

22.

[Menu bar --> Add --> Vision sensor --> Perspective type] 來打開 vision sensor ，將position and orientation 設置為 (0,0,0)，再來將Far clipping plane 設置為1，Resolution X / Y 設置為 256 / 256 

23.

進入 Show filter dialog ， 選擇 Edge detection on work image 然後點 Add filter 來新增，並按旁邊向上的箭頭讓他跑到中間，然後點兩下來改 Threshold = 0.2  。

24.

在 scene 上 右鍵 add floating view ，然後在浮動視窗上面 選 view > Associate view with selected vision sensor ，就可以模擬了。

25.
